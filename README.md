# 📄 AI Resume Screening System

An **AI-powered Resume Screening System** that automatically analyzes and ranks resumes using **Machine Learning and Natural Language Processing (NLP)** techniques.
This system helps recruiters quickly identify the most suitable candidates by extracting skills and matching resumes with job requirements.

---

## 🚀 Features

* 📑 **Resume Parsing** – Extracts important information from resumes
* 🧠 **Machine Learning Model** – Classifies resumes based on job roles
* 🔍 **Skill Extraction** – Identifies key skills using NLP techniques
* 📊 **Candidate Ranking** – Filters and ranks resumes automatically
* 🌐 **Web Interface** – Simple interface to upload and analyze resumes

## ⚙️ How It Works

1. Upload a resume file.
2. The system extracts text from the resume.
3. NLP techniques process the text.
4. Machine learning model analyzes the resume.
5. The system predicts the **most suitable job category**.

## ▶️ Installation

Clone the repository:

```
git clone https://github.com/sanketsawant72/resume-screening.git
```

Navigate to the project folder:

```
cd resume-screening
```

Install required libraries:

```
pip install -r requirements.txt
```

Run the application:

```
python app.py
```

Open in browser:

```
http://127.0.0.1:5000
```
